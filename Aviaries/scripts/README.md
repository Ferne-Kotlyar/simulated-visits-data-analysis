## Scripts

- **aviaries_data_analysis.R:** Script to analyse and make a figure for the aviary experiment. In this experiment, conducted in February and March, 2018 and 2019, there are 3 treatments: hand pollination (HP), live Green Hermit hummingbird pollination (GREH), and live Rufous-tailed hummingbird pollination (RTAH).
- **nectar_removal.Rmd:** Script to analyse and make a figure for the nectar extraction experiment. This script calls the aviaries_data_analysis.R script, so only this script needs to be run. It also combines the aviary and nectar extraction figures to make a single figure. In this experiment, conducted in February and March 2017-2019, there are 2 treatments: hand pollination (HP) and nectar extraction (HPNE).
