# Scripts README: Nectar Dye Experiment

**Created by:** Ferne Kotlyar \| **Date created:** October 20, 2025

**Updated by:** Ferne Kotlyar \| **Date updated:** May 07, 2026

## Scripts

- **Nectar_Dye_Data_Cleaning.R:** Script to clean the raw data and prepare it for analysis. Data files were used from the rawdata folder.

- **Nectar_Dye_Figures.Rmd:** Notebook that produces Fig.3 in the "Nectar deposition on stigma links trait matching to plant reproductive success" manuscript.

- **Nectar_Dye_Stats.Rmd:** Notebook that runs the statistical analyses described in the statistical analysis section based on "2.2 Specialist bills deposit nectar on stigma" and Methods "4.4.2 Nectar dye experiment" in the above manuscript.
