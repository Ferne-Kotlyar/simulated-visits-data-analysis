## Scripts

- **Fruit_Set_Data_Cleaning.R:** Script to clean the raw data and prepare it for fruit set analysis. Data files were used from the rawdata folder.

- **Simulated_Visits_2024_Figures.Rmd:** Notebook that produces Fig. 4 in the "Nectar deposition on stigma links trait matching to plant reproductive success" manuscript.

- **Simulated_Visits_2024_Stats.Rmd:** Notebook that runs the statistical analyses described in the statistical analysis section based on "2.4 Pollination requires nectar and bill" and Methods "4.6 Mechanism of simulated visits" in the above manuscript.
